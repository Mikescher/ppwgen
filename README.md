# ppwgen

Create pronouncable passwords from the commandline

`````
./ppwgen - PronouncablePasswordGenerator.

# (forked from KeePass PronouncablePassword plugin)

Usage: ppwgen
       ppwgen <minlength>
       ppwgen [minlength] --count=<c>
       ppwgen --version
       ppwgen --help

Options:
  --length=<len>
  --digits
  --no-digits
  --mc,  --mixed-case
  --uc,  --upper-case
  --lc,  --lower-case
  --rc,  --random-case
  --rmc, --random-mixed-case
  --mp,  --more-pronouncable
  --symbols
  --count=<c>
  --seed=<s>
`````
